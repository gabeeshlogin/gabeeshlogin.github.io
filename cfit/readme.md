# EMPOWER by Christina Sarafian

Static multi-page site (plain HTML + CSS + vanilla JS, no build step) for
**EMPOWER** — the rebrand of CFIT Personal Training. Deploys on GitHub Pages
at `/cfit/` using relative paths, so it is portable to Christina's own domain.

## Pages
- `index.html` — homepage
- `about.html` — about Christina
- `services.html` — services
- `booking.html` — free-consultation booking (SimplyBook.me embed)
- `contact.html` — contact form (mailto) + direct contact

## Shared assets
- `style.css` — monochrome-luxury design system (warmed to cream)
- `main.js` — interactions + GSAP/ScrollTrigger motion (loaded from cdnjs)

See the PR description for the full client setup checklist (SimplyBook.me URL,
photography, going live / removing `noindex`).
